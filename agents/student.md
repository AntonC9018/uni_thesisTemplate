## When assisting the student in writing their thesis

- Prioritize using existing commands over built-in latex commands.
  Scan for these in the template of the user's language.
- If the student asks you to review their thesis,
  suggest these commands if their raw latex equivalents are used.
- In cases where the template has extensions over the official style guide that the student may provide,
  or has minor typesetting discrepancies, like the format of lists or code listings,
  assume the template wins.
  Do consult the guide for page counts, wording suggestions, main chapter distributions, etc.
- Thesis writing should be scoped to `main.tex` and any files added by the student.
  `config.sty` is not one of them. Never edit it unless explicitly asked to ignore this rule.
- Syncing the student's configuration with the upstream upon request is OK.
  If you do, make sure the student's changes are in git and then edit their thesis 
  to comply with the updated API's.
  The PR might include relevant migration information.
- Suggest updating the AI usage section if relevant.
- Suggest explicitly marking unedited fragments fully generated by AI in text.
- Unlabeled figures/tables/code are not allowed in the main text. Always use the labeled variants.
- Do not insert company names and such as author names when editing the bibliography.
  Try to find the author name in the linked page, and omit the author field if it's not available.

## When converting to latex from a different text format:

- Keep the original comments around the metadata section of the template.
- Silently insert captions if they are missing in the source but are required.
- Silently format inline code as such. For this, prefer commands mentioned in the template.
- Edit the bibliography file, pasting in the new sources.
- Never change the text size - any text size other than the template default for a given context is wrong.
- Document being too long after conversion is totally ok, you should not silently change 
  the document content or disregard previous instructions to make it fit the required number of pages.
  You may compress text only upon an explicit request.
- Unreferenced bibliography or not enough sources is also completely fine after the conversion.
